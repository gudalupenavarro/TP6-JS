3) ¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.

Manejo de variables: Son un elemento que tiene un nombre determinado y un valor asociado. Los nombres de estas deben comenzar con una letra y puede ir seguida de letras, numeros o el simbolo "_". Las variables, al no estar asociadas a ningun valor, nos dan la posibilidad de asignarle o reasignarle nuevos valores de todos los tipos.Esto debido a lo debilmente tipado y dinamico que es JavaScript (esto quiere decir que de una variable que tiene un dato numerico puede pasar a almacernar un dato de tipo booleano)
Los nombres de las variables deben ser simbolicos -Existen tres tipos:

  • var
  • let
  • const

Manejo de tipo de datos: En el ultimo estandar de ECMAScript, se definen nueve tipos de datos.

  • Undefined
  • Boolean
  • Number
  • String
  • BigInt
  • Symbol
  • Nul
  • Object
  • Function

Manejo de las expresiones regulares: Sistema de busqueda captura o reemplazo de texto utilizando patrones, que nos dan la posibilidad de una busqueda de texto de una forma relativamente sencilla y abstracta. Estos son representados mediante una cadena de texto, donde ciertos simbolos tienen un significado especial. En JavaScript, las expresiones regulres tambien son objetos. Los métodos que funcionan con expresiones regulares en JavaScript son las siguientes: regexp.exec, regexp.test, string.match, string.replace, string.search, y string.split .

4) ¿Cómo son todas las formas de declaracion de funciones en JS?.

Hay 7 formas de declarar una funcion en JS:

  • Function declaration:

Siendo esta, la mas sencilla de todas. Para crearla, necesitaremos la palabra clave function, un nombre que identifique a la funcion y los parametros a utilizar deben estar entre parentesis o llaves (aunque los parametros no son obligatorios)

  • Function expression:

Esta funcion es parecida a la anterior, solo que esta no comienza con la palabra function, y el nombre de la funcion es opcional. Estas funciones se pueden almacenar en variables

  • IIFE(Immediately Invoked Function Expression:

Esta funcion nos sera util en el caso de querer utilizarla una sola ocasion. Este tipos de funciones se ejecutan inmediatamente y no son accesibles posteriormente.

  • Shorthand method definition:

Metodo de declarar de un objeto o en las clases de ES6. Para crear este tipo de función debemos darle un nombre de función seguido de una lista de parámetros entre paréntesis y los símbolos de llaves para delimitar el cuerpo de las instrucciones.

  • Arrow Function:

Esta función no crea su propio contexto al ejecutarse, son Anonimas. En las Arrow function el objeto arguments, no se encuentra en el contexto de la función. Por ultimo, si al definir la funcion no utilizamos el simbolo de las llaves, la funcion devolvera como resultado de la funcion el resultado de la ejecución de la instrucción que hayamos indicado.

  • Generator function

La caracteristica que destaca de esta funcion, es que nos permite salir o parar la funcion dentro de del conjunto de instrucciones que conforma la funcion para luego retornar la ejecución desde el punto donde la paramos. La sintaxis es parecida a las otras, solo que se utiliza el simbolo * al comienzo de la funcion.

  • Function constructor:

Esta, es una forma inusual de declarar variables, ya que cre una variable que invoque a un objeto Function, y de esta manera, enviar todos los argumentos que deseemos. Los primeros argumentos seran los parametros, y el utlimo argumento sera el codigo.

6) ¿Cuales son las estructuras de control de JS?.

Las estructuras de control son las siguientes:

IF

Esta estructura es de tipo condicional, es decir, que si la condicion es cierta el bloque de codigo se ejecutara (una vez), mientras que si no, no pasara nada, o saltará al else.

ELSE

El else, es una estructura, que va en conjunto con el IF, aunque no siempre es obligatorio. Este bloque de codigo sera el que se ejecute en caso de que la condicion sea falsa.

SWITCH

Nos da la posibilidad de ejecutar diferentes instrucciones basadas en diferentes condiciones Le damos un valor, y este compureba si ese valor coincide con alguno de los casos.

FOR

Esta estructura bucle for, nos da la posibilidad de ejecutar el codigo las veces que especifiquemos

WHILE

Esta estructura en bucle, se ejecutara mientras la condicion que indicamos sea verdadera.

DO

A diferencia del While, este bucle siempre se ejecutara una primera vez. Solo se volvera a ejecutar en caso de que la condicion siga siendo cierta.

7) ¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?.

El objeto DOM (Document Object Model, es una interfaz de programacion que facilita una representacion estructurada del documento (modelizan todos los elementos que pueda tener dentro de la pagina) y por medio de JavaScript que nos permitirá crear, remover o cambiar elementos del documento.
